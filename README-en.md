# Epic Space Battles - Movie Universe

## 📌 Project Description
* **Section Name:** Space Battles & Cinematic Strategy.
* **Objective:** To provide an immersive experience where users can explore historical space battles from cinema, watch trailers, and check live reviews.
* **Target Audience:** Science fiction fans, movie enthusiasts, and users interested in space strategy simulations within an academic and technological context.

## 🚀 Implemented Features
* **Battle Listing:** A dynamic page featuring the most iconic battles (*Star Trek, Star Wars, Guardians of the Galaxy, and Avengers: Endgame*).
* **Detail Pages:** Individual pages with movie descriptions, embedded trailers, and dynamic movie data.
* **API Integration:** Dynamic data fetching for IMDb ratings and vote counts, automatically generating a 5-star visual rating system.

## 🛠️ Technologies Used
* **HTML5:** Semantic structured pages for listing and movie details.
* **CSS3:** Advanced styling featuring dark-mode aesthetics and space-themed visual intensity.
* **JavaScript:** Logic for asynchronous requests (Fetch API), dynamic DOM manipulation, and visual stars calculation.
* **OMDb API:** External REST API used to consume real-time movie statistics from IMDb.
