# Batalhas Espaciais Épicas - Universo Cinematográfico

## 📌 Descrição do Projeto
* **Nome da seção:** Batalhas Espaciais e Estratégia Cinematográfica.
* **Objetivo:** Proporcionar uma experiência imersiva onde os usuários podem explorar batalhas históricas do cinema, assistir trailers e conferir avaliações atualizadas.
* **Público-alvo:** Fãs de ficção científica, entusiastas de cinema e usuários interessados em simulações de estratégias espaciais em um contexto acadêmico e tecnológico.

## 🚀 Funcionalidades Implementadas
* **Listagem de Batalhas:** Página dinâmica com as batalhas mais icônicas (*Star Trek, Star Wars, Guardiões da Galáxia e Vingadores: Ultimato*).
* **Páginas de Detalhes:** Páginas individuais com descrições, trailers integrados e exibição de dados dinâmicos.
* **Integração com API:** Consumo de dados em tempo real da nota do IMDb e contagem de votos, gerando automaticamente um sistema visual de avaliação por estrelas.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Estrutura semântica das páginas de lista e detalhes.
* **CSS3:** Estilização avançada com temática dark e efeitos visuais espaciais.
* **JavaScript:** Lógica para requisições assíncronas (Fetch API), manipulação dinâmica do DOM e cálculo do preenchimento de estrelas.
* **OMDb API:** API externa utilizada para consumir dados oficiais de classificação de filmes.

## 👥 Organização da Equipe
Projeto desenvolvido em equipe composta por **5 pessoas**.
* **Divisão de Tarefas:**
    * Design e estruturação colaborativa, onde cada integrante realizou o desenvolvimento completo de uma página do projeto.

## ⚠️ Desafios Enfrentados e Melhorias Futuras
* **Desafios Técnicos:** Sincronizar o projeto e subir os arquivos corretamente no GitHub, além de implementar as funcionalidades específicas em JavaScript, como tratar os retornos da API.
* **Melhorias Futuras:** Com mais tempo, implementaríamos uma aba dedicada inteiramente a séries que possuem batalhas épicas, expandindo o catálogo além dos filmes.
